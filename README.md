**# Airline Management System**

A desktop-based Java application for managing airline operations, including bookings, customer management, and flight details.

**## Feature**
- **User Login:** Secure login for staff or administrators.
- **Book Flight:** Book tickets for customers with PNR generation.
- **Journey Details:** View booking details using PNR.
- **Cancel Tickets:** Cancel reservations and update records.
- **Boarding Pass:** Generate and view boarding passes.
- **Customer Management:** Add and view customer information.
- **Database Integration:** Uses MySQL for storing customer and booking data.

**## Technologies Used**

- Java (Swing for GUI)
- MySQL Database
- JDBC for database connectivity
- [JCalendar](https://toedter.com/jcalendar/) for date selection
- [RS2XML](https://sourceforge.net/projects/finalangelsanddemons/) (DbUtils) for displaying ResultSet in JTable
